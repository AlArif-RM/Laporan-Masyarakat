# Logo Provinsi

Folder ini dipakai untuk logo pada PDF laporan dan PDF rekap untuk admin provinsi.

## Format nama file

- Gunakan kode provinsi sebagai nama file.
- Ekstensi yang didukung:
  - `.png`
  - `.jpg`
  - `.jpeg`
  - `.webp`

Contoh:

- `12.png` untuk Provinsi Sumatera Utara
- `31.png` untuk DKI Jakarta

## Referensi

Lihat aturan akun nasional di `docs/admin-wilayah-nasional.md`.

## Import ZIP

Anda juga bisa import massal dari halaman `Logo Wilayah`.
Nama file di dalam ZIP tetap harus mengikuti kode wilayah, misalnya `provinces/12.png`.
