# Logo Kabupaten dan Kota

Folder ini dipakai untuk logo pada PDF laporan dan PDF rekap untuk admin kabupaten/kota.

## Format nama file

- Gunakan kode kabupaten/kota sebagai nama file.
- Ekstensi yang didukung:
  - `.png`
  - `.jpg`
  - `.jpeg`
  - `.webp`

Contoh:

- `12.07.png` untuk Kabupaten Deli Serdang
- `12.71.png` untuk Kota Medan

## Referensi

Lihat aturan akun nasional di `docs/admin-wilayah-nasional.md`.

## Import ZIP

Anda juga bisa import massal dari halaman `Logo Wilayah`.
Nama file di dalam ZIP tetap harus mengikuti kode wilayah, misalnya `regencies/12.07.png`.
