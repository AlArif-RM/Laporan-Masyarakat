# Logo Kecamatan

Folder ini dipakai untuk logo pada PDF laporan dan PDF rekap untuk admin kecamatan.

## Cara pakai

1. Siapkan file logo resmi kecamatan.
2. Ganti file sesuai kode kecamatan.
3. Tidak perlu ubah kode aplikasi.

## Format nama file

- Gunakan kode kecamatan sebagai nama file.
- Ekstensi yang didukung:
  - `.png`
  - `.jpg`
  - `.jpeg`
  - `.webp`

Contoh:

- `12.07.26.png` untuk Kecamatan Percut Sei Tuan
- `12.07.28.png` untuk Kecamatan Lubuk Pakam

## Referensi

Lihat aturan akun nasional dan format logo di `docs/admin-wilayah-nasional.md`.

## Import ZIP

Anda juga bisa import massal dari halaman `Logo Wilayah`.
Nama file di dalam ZIP tetap harus mengikuti kode wilayah, misalnya `districts/12.07.26.png`.
